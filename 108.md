# 108、最大公因數計算

## 求最大公因數

使用輾轉相除法。

GCD(18, 24)

=> 18 % 24 = 18, 18 != 0

=> GCD(24, 18)

=> 24 % 18 = 6, 6 != 0

=> GCD(18, 6)

=> 18 % 6 = 0

=> 6 (the answer)

## 遞迴函數解法

```
static int GCD(int m, int n)
{
    if (m % n == 0) {
        return n;
    }
    return GCD(n, m % n);
}
```

        static int findCommon(int m, int n)
        {
            while (m % n != 0)
            {
                int t = n;
                n = m % n;
                m = t;
            }
            return n;
        }
    }
